# Image Background Remover - MVP 需求文档

## 1. 项目概述

### 1.1 项目名称
**Image Background Remover**

### 1.2 项目目标
做一个可快速上线的在线抠图网站，让用户上传图片后，自动去除背景，并可直接下载透明背景图片。

### 1.3 MVP 目标
验证这三件事：

1. 用户是否有真实使用需求
2. Cloudflare + Remove.bg API 方案是否可行
3. 网站是否具备后续 SEO 和商业化空间

---

## 2. 产品定位

### 2.1 核心定位
一个**极简、开箱即用**的在线抠图工具网站。

### 2.2 核心卖点
- 无需安装软件
- 上传即抠图
- 结果直接预览
- 一键下载透明背景图

### 2.3 目标用户
第一阶段优先覆盖：

- 电商卖家
- 自媒体创作者
- 设计需求较轻的普通用户
- 需要快速处理头像/商品图的人

## 4. 核心用户流程

### 4.1 主流程
1. 用户进入首页
2. 用户上传图片
3. 前端将图片提交到后端接口
4. 后端转发给 Remove.bg API
5. 返回去背景结果
6. 前端展示处理前后效果
7. 用户点击下载

### 4.2 异常流程
#### 上传失败
- 提示：上传失败，请重试

#### 文件格式不支持
- 提示：仅支持 JPG / PNG / WebP

#### 文件过大
- 提示：图片过大，请上传 10MB 以内文件

#### API 处理失败
- 提示：处理失败，请稍后再试

## 6. 功能需求

### 6.1 图片上传
#### 要求
- 支持点击上传
- 支持拖拽上传
- 支持格式：
  - JPG
  - PNG
  - WebP

#### 限制
- 单张图片最大 10MB

### 6.2 去背景处理
#### 要求
- 通过后端调用 Remove.bg API
- 不在前端直接暴露 API Key
- 接口处理完成后返回图片结果

#### 处理方式
- 图片只在请求生命周期内存在内存中
- 不做落盘
- 不做对象存储

### 6.3 结果展示
#### 要求
- 展示原图
- 展示去背景结果
- 提供下载按钮

#### 可选增强
- 左右对比滑块（非必须）
- 棋盘格透明背景（建议有）

### 6.4 下载
#### 要求
- 支持浏览器直接下载
- 默认文件名可自动生成，例如：
  - `removed-background.png`

### 6.5 错误提示
#### 要求
用户能明确知道失败原因，至少区分：
- 格式不支持
- 文件过大
- 网络错误
- API 失败

---

## 7. 非功能需求

### 7.1 性能
- 常规图片处理响应尽量控制在可接受范围内
- 用户在等待时必须看到 loading

### 7.2 安全
- Remove.bg API Key 仅保存在服务端环境变量
- 不保存用户图片
- 不返回多余敏感信息

### 7.3 可维护性
- 前后端代码结构清晰
- 环境变量配置明确
- README 可指导部署

---

## 8. 技术方案

### 8.1 前端
#### 推荐
- Next.js

#### 原因
- 适合做 SEO 页面
- 部署到 Cloudflare 方便
- 后续扩展博客/落地页方便

### 8.2 后端
#### 推荐
- Cloudflare Workers

#### 职责
- 接收上传
- 调用 Remove.bg API
- 返回处理结果

### 8.3 部署
#### 推荐
- 前端：Cloudflare Pages
- API：Cloudflare Workers

### 8.4 存储
#### 本期方案
- **不使用持久存储**
- 图片仅做请求级处理
- 不保存在数据库或对象存储中

---

## 9. API 需求

### 9.1 去背景接口
#### 建议接口
`POST /api/remove-background`

#### 请求
- `multipart/form-data`
- 字段：
  - `image`

#### 返回
- 成功：去背景后的图片二进制
- 失败：JSON 错误信息

#### 错误结构示例
```json
{
  "error": "Image too large"
}
```

## 12. 风险与限制

### 12.1 外部 API 依赖
- Remove.bg 费用和稳定性受第三方影响

### 12.2 Cloudflare 限制
- 请求体大小
- 执行时长
- 大文件处理能力有限

### 12.3 无存储方案限制
- 无历史记录
- 页面关闭即无数据
- 不支持回查
